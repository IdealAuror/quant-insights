## 核心机制（双仓结构）

### 主仓（80%资金）——年线策略

| 项目 | 参数 |
|------|------|
| 买入条件 | 年线偏离度 ≤ -1%（价格低于250日均线1%以上） |
| 卖出条件 | 年线偏离度 ≥ +8%（价格高于250日均线8%以上） |
| 建仓方式 | 每次买入总资金的20%（80%÷4次），分4批买入 |
| 卖出方式 | 每次卖出当前主仓的30%，最多卖3次 |
| 底仓保护 | 保留主仓资金的10%（即总资金的8%）作为永久底仓 |
| 买入间隔 | 7天（两次买入至少间隔7个交易日） |

**逻辑：** 价格低于年线1%以上时，市场处于相对低估区间，分批建仓；价格高于年线8%以上时，市场处于相对高估区间，分批止盈。

> ⚠️ **关键澄清：** 卖出条件的"8%"是指价格偏离年线的幅度，而非持仓盈亏。即 `(价格 - 年线) / 年线 > 0.08`，与买入成本无关。

### 辅仓（20%资金）——RSI策略

| 项目 | 参数 |
|------|------|
| 买入条件 | RSI(14) ≤ 30 且 价格 < MA20（同时满足） |
| 卖出条件 | RSI峰值超过70后，从峰值回落≥5个点 |
| 兜底止损 | RSI ≤ 20 时无条件清仓 |
| 建仓方式 | 一次性买入总资金的20% |
| 卖出方式 | 一次性清仓 |

**逻辑：** 当RSI进入超卖区（≤30）且价格低于短期均线时，市场短期超跌，一次性买入辅仓。买入后追踪RSI峰值，当峰值超过70后出现回落（≥5个点），说明短期超买情绪释放，一次性止盈。极端情况下RSI跌破20时无条件止损。

## 实盘适配建议

### 适合人群
- 稳健型投资者，追求绝对收益而非暴利
- 不想天天盯盘，希望账户波动小
- 把A股作为资产配置的一部分
- 追求分红现金流 + 资本增值的双重回报

### 不适合人群
- 追求年化15%+的激进投资者
- 想在牛市初期跑赢大盘的投资者
- 无法接受低频交易的投资者

### 实盘操作要点
1. **资金门槛：** 建议10万以上（否则佣金最低5元侵蚀收益）
2. **交易时间：** 建议14:50或收盘集合竞价
3. **心态管理：** 预期年化5%~9%（含分红再投），最大回撤≤11%
4. **分红处理：** 务必选择"分红再投"，这是年化从3%~6%提升到5%~9%的关键
5. **红利税提醒：** 场内ETF分红通常按10%缴税（持有满1年可申请退税），实盘收益可能比回测低约0.5%~1.5%/年
