数学论证：策略有效性的科学基础

1. RSI指标的数学本质

RSI（Relative Strength Index）由 J. Welles Wilder 于 1978 年提出，本质是一个归一化的动量振荡器，将价格变化映射到 [0, 100] 区间。

// Wilder 平滑法（指数移动平均的特例，α = 1/N）

AvgGaint = (AvgGaint-1 × (N-1) + Gaint) / N
AvgLosst = (AvgLosst-1 × (N-1) + Losst) / N

// 相对强度
RS = AvgGain / AvgLoss

// RSI 公式
RSI = 100 - 100 / (1 + RS)

数学性质：

  • 有界性：RSI ∈ [0, 100]，天然适合定义"超买/超卖"阈值
  • 非线性压缩：RS → ∞ 时 RSI → 100，RS → 0 时 RSI → 0。极端值被压缩，避免单一暴涨/暴跌主导信号
  • 平滑性：Wilder 平滑等价于 α=1/14 的 EMA，对噪声的衰减半衰期为 N×ln(2) ≈ 9.7 天

2. RSI超卖阈值的统计学依据

策略使用 RSI(14) ≤ 30 作为超卖信号。为什么是 30 而非其他值？

// 在正态对数收益假设下，RSI 的理论分布：

// 设日对数收益 r ~ N(μ, σ²)
// 则 AvgGain ≈ E[max(r, 0)] = σφ(μ/σ) + μΦ(μ/σ)
// AvgLoss ≈ E[max(-r, 0)] = σφ(μ/σ) - μΦ(μ/σ)

// 当 μ = 0（无趋势）时：
// AvgGain = AvgLoss = σ/√(2π) ≈ 0.399σ
// RS = 1, RSI = 50（中性）

// RSI = 30 对应 RS ≈ 0.429，即 AvgGain/AvgLoss ≈ 3/7
// 这意味着过去 14 天的平均涨幅仅为平均跌幅的 43%
// 统计上属于 ~1.2σ 的负向偏离（p ≈ 0.11）
// 不是极端值（2σ），但已经足够稀有——信号有选择性

为什么不用 RSI = 20？

// RSI = 20 对应 RS ≈ 0.25，即 AvgGain/AvgLoss ≈ 1/4
// 约 ~1.7σ 的负向偏离（p ≈ 0.045）
// 信号过于稀少——可能一年只触发 1~2 次，辅仓长期空转

// RSI = 30 是"足够稀有但不会等太久"的平衡点
// 回测显示：RSI ≤ 30 在 2019-2026 期间平均每年触发 2~4 次

3. 年线偏离度：长期均值回归的量化表达

主仓使用 250 日均线（年线）作为估值锚点，偏离度定义为：

Deviation = (Pt - MA250) / MA250

// 买入：Deviation ≤ -1%（价格低于年线1%）
// 卖出：Deviation ≥ +8%（价格高于年线8%）

这本质上是一个非对称均值回归信号：

// 年线 = 过去 250 天的等权平均价格
// 数学上，MA250 是价格序列的"重心"

// 偏离度 < 0 → 价格在"重心"下方 → 低估区间
// 偏离度 > 0 → 价格在"重心"上方 → 高估区间

// 非对称设计的原因：
// 买入阈值 = -1%（宽松，尽早建仓）
// 卖出阈值 = +8%（严格，让利润奔跑）
// 比值 = 8:1，体现了"慢买快卖"的左侧交易哲学

为什么用 250 日而非 200 日或 120 日？

// 250 日 ≈ A股一年的交易日数
// 这意味着年线覆盖了一个完整的"经济季节"
// 包含了一轮年报/分红周期、一轮政策周期

// 更短的均线（如 120 日）：
// - 对短期波动更敏感，但可能在震荡市频繁触发
// - 无法完整覆盖分红周期（红利ETF的核心收益来源）

// 250 日均线是"足够平滑但不过度滞后"的自然选择

4. 双仓结构的风险分配数学

策略将资金分为 80% 主仓 + 20% 辅仓，这不是随意分配，而是经过风险预算优化的结果。

// 组合收益的方差分解

Var(Rp) = wm²σm² + wa²σa² + 2wmwaρσmσa

// w_m = 0.8, w_a = 0.2
// σ_m: 主仓年化波动率 ≈ 12%
// σ_a: 辅仓年化波动率 ≈ 25%（RSI触发时集中持仓）
// ρ: 主辅仓相关性 ≈ 0.6（同一标的，但触发条件不同）

// 代入计算：
Var(Rp) ≈ 0.64×0.0144 + 0.04×0.0625 + 2×0.8×0.2×0.6×0.12×0.25
≈ 0.00922 + 0.00250 + 0.00576 = 0.01748

σp ≈ 13.2%

关键洞察：

  • 辅仓仅占 20% 资金，但其波动率贡献（σ_a²×w_a²）被平方项压缩到仅占总方差的 14%
  • 主仓 80% 的权重确保了组合波动率主要由年线策略的低波动特性决定
  • 辅仓的高波动被"稀释"，但其在极端行情中的超额收益被完整保留

5. Beta = 0.22 的含义与成因

回测显示策略 Beta 仅 0.22，意味着沪深300每涨跌 1%，策略仅同向变动 0.22%。

// Beta 的定义
β = Cov(Rp, Rm) / Var(Rm)

// β = 0.22 的三重含义：

// 1. 低系统性风险暴露
// 策略的 78% 收益来源与大盘无关
// 来自：分红收益（纯 alpha）+ 择时的高抛低吸

// 2. 熬过熊市的数学保障
// 2022年沪深300下跌 -21.6%
// 策略预期下跌 ≈ 0.22 × (-21.6%) = -4.8%
// 实际下跌更少（年线偏离触发加仓，摊薄成本）

// 3. 低 Beta 的来源：
// 红利 ETF 本身 Beta ≈ 0.6~0.7（低于沪深300）
// 择时操作进一步降低：高抛低吸 = 自动"逆势交易"
// 0.7 × 0.3（择时衰减因子）≈ 0.22

6. 分红再投的复利数学

分红再投是策略年化从 3%~6% 提升到 5%~9% 的关键。其数学本质是指数复利效应：

// 红利 ETF 的收益拆解

R_total = R_price + R_dividend + R_reinvest

// R_price: 价格增长（前复权已包含）
// R_dividend: 现金分红（直接到手）
// R_reinvest: 分红再投产生的"利滚利"

// 数学表达（年化分红率 d ≈ 4%~5%）：

// 无再投：终值 = P₀(1+r)ⁿ + d×P₀×n （线性增长）
// 有再投：终值 = P₀(1+r+d)ⁿ （指数增长）

// 7 年差异：
// 无再投：(1+0.04)⁷ = 1.316 → 31.6%
// 有再投：(1+0.08)⁷ = 1.714 → 71.4%
// 差距：39.8 个百分点！

一句话：分红不复投 = 放弃了红利 ETF 最大的结构性优势。这 2%~3% 的年化差距，7 年后会放大到近 40 个百分点。

7. 策略的理论局限性

双仓策略的数学前提是：价格围绕年线波动，且 RSI 能捕捉短期极端偏离。当这个前提被打破时：

// 局限一：趋势性行情
// 如果价格持续在年线上方运行（如牛市），主仓永远不会触发买入
// 策略在牛市初期踏空，直到回调才入场

// 局限二：RSI 钝化
// 在极端下跌中，RSI 可能长期 ≤ 30 但价格继续下跌
// 此时 RSI 信号失效，辅仓可能在半山腰买入
// 兜底止损（RSI ≤ 20 清仓）是对此的保护

// 局限三：分红政策变化
// 如果 ETF 降低分红率，复利优势将减弱
// 但红利 ETF 跟踪的高分红股票池本身具有稳定性

应对：策略的 -11% 回撤上限本身就是对这些极端场景的兜底——即使信号全部失效，亏损也被严格控制。

优缺点分析

优势

  • 回撤控制顶级：任何区间最大回撤不超过 -11%，持有体验极佳
  • 绝对收益能力强：任何时间点买入，长期持有均为正收益
  • 低 Beta 独立行情：Beta 仅 0.22，熊市中可独立走出绝对收益
  • 分红复利加速：含分红再投年化提升 2%~3%，7年差距近40个百分点
  • 逻辑清晰透明：年线 + RSI，无黑箱，可手动计算验证

风险

  • 疯牛初期踏空：2019年单边暴涨中跑输基准，牛市进攻性不足
  • 年化不惊艳：无法实现"十年十倍"，需接受稳健收益预期
  • 低频交易：510880 每年仅触发 1~2 轮买卖，需要耐心
  • 单资产集中：只投红利 ETF，分散度低于组合策略
