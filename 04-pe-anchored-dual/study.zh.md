## 核心机制（三层架构）

### 第一层——PE仓位中枢（绝对估值锚定）

| PE区间 | 权益上限 | 单池底仓 | 状态 |
|--------|---------|---------|------|
| PE < 13 | 95% | 40% | 低估满仓 |
| 13 ≤ PE < 15 | 95% → 65%（线性） | 40% → 20%（线性） | 中等偏高 |
| 15 ≤ PE < 17 | 65% → 30%（线性） | 20% → 10%（线性） | 高估减仓 |
| PE ≥ 17 | 30% | 10% | 高估轻仓 |

**逻辑：** 沪深300 PE历史均值约12.5~13倍，以此为锚。PE<13视为低估，满仓95%；PE>17视为高估，轻仓30%。线性过渡，无突变。避免相对分位数的滞后性。

### 第二层——动量分配（资金倾斜）

比较红利ETF与创业板ETF过去20日涨幅，强者获60%权益资金，弱者获40%。在趋势明确时提升收益，且不增加总体仓位。

### 第三层——双池独立运行

#### 价值池（红利ETF 510880）——网格定投 + RSI辅仓

| 项目 | 参数 |
|------|------|
| 买入条件 | 价格不高于年线8%以内（dev < +8%） |
| 卖出条件（第一档） | 偏离年线 > +18%（止盈） |
| 卖出条件（第二档） | 偏离年线 > +30%（清仓至底仓） |
| 建仓方式 | 分4批买入，每批25%×0.8 折价 |
| 卖出方式 | 每次卖30%仓位，最多3次 |
| PE<15时日频买入 | 每天一次（低估加速建仓） |
| PE≥15时 | 隔3天一次 |

**辅仓（RSI超卖）：**
| 项目 | 参数 |
|------|------|
| 买入条件 | RSI(14) ≤ 30 且 价格 < MA20 |
| 卖出条件 | RSI峰值 ≥ 70 且 从峰值回落 ≥ 5点 |
| 兜底止损 | RSI ≤ 20 |
| 仓位 | 20% 主仓额度 |

#### 成长池（创业板ETF 159915）——趋势跟随

| 项目 | 参数 |
|------|------|
| PE<15时买入 | 价格 > 年线×0.95 且缩量（vol < vol_ma20×1.2） |
| PE≥15时买入 | 价格 > 年线×1.02 |
| 止损 | 从持有期最高点回撤 > 8% 或 跌破年线×0.92 |
| PE≥15且持仓 | 强制清仓至底仓 |

## 收益来源归因

- **Beta贡献：** 0.82 × 57.73% ≈ 47.3%
- **Alpha贡献：** 150.62% − 47.3% ≈ **103.3%**
- **Alpha来源：** 网格波段、动量倾斜、底部左侧建仓
- **实际Alpha（官方）：** 0.08——大部分超额来自风险调整后的择时与资产配置，而非选股

## 优缺点分析

### 优势

| 优势 | 说明 |
|------|------|
| 不对称Alpha | 熊市抗跌（2022年仅-8%）、牛市跟上（2019-2021超额稳定）、震荡市网格增厚 |
| PE锚定客观 | 绝对估值中枢，避免相对分位数的滞后性 |
| 双资产互补 | 价值（红利）提供底仓稳定，成长（创业板）提供弹性 |
| 动量倾斜有效 | 在不增加总仓位的前提下提升资金效率 |
| 硬性底仓保护 | 即使PE>17，保留20%底仓避免完全踏空 |
| 逻辑清晰 | PE+动量+网格+RSI，每层独立可验证 |

### 劣势

| 劣势 | 说明 |
|------|------|
| 回撤绝对值较高 | 26%回撤对保守型投资者仍有压力，可叠加趋势过滤器 |
| 参数依赖PE阈值 | 若A股估值中枢长期漂移（如利率环境改变），PE阈值需重设 |
| 极端流动性风险 | ETF交易量充足，但连续跌停下滑点会放大回撤 |
| 分仓管理复杂 | 双池三标的，比单资产策略需要更多操作 |

## 实盘适配建议

### 适合人群
- 进攻型投资者，可接受20%+回撤
- 相信A股估值均值回归的长期投资者
- 希望在熊市中大幅跑赢大盘的投资者

### 不适合人群
- 无法接受20%+回撤的保守投资者
- 追求极简策略（单标的一键操作）的投资者

### 实盘操作要点
1. **分仓管理：** 实盘先以50%资金运行，观察3个月，净值稳定后再加仓
2. **增加波动率过滤器（可选）：** 当市场波动率（20日年化）> 35%时，将PE满仓阈值从13下调至12
3. **成长止损放宽（可选）：** 强趋势行情中将止损从8%上调至12%，减少中途下车
4. **定期再平衡：** 每季度检查PE阈值是否仍适用（参考10年期国债收益率变化）
5. **交易时间：** 建议14:50执行，减少盘中波动干扰
